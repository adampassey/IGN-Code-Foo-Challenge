3) What is the minimum number of moves required for a knight to cover the entire chess board? Write a program to prove it. Provide a graph with the move number on the x axis and the number of squares on the y axis.

63 Moves

The minimum number of moves required for a knight to cover an entire chessboard is 63.

This is assuming the size of the chessboard is a typical 8x8, and the knight must move according to the rules of chess.

Initially, I intended to create a brute force program that would continually try new, slightly varied routes until finding a route that touched every tile in only 63 moves. I shortly realized that the sheer number of route possibilities would complicate things. It may take hours or even days for a brute force program to solve this kind of puzzle. I wanted to be able to show you (the IGN Code Foo Challenge Judge) that I could write a program that could logically find a route that could touch every tile in just 63 moves- starting from any tile. The success rate has been averaged at approximately 98%.

This demonstration was built using the HTML5 Canvas tag and an in-development javascript-based game platform that I'm working on. And yes, I threw in some particle effects just for fun. Compatible with the following browsers:
